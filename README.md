# Photo3D — 3D-моделирование по фото

Сервис создания 3D-моделей по фотографиям людей с ручной раскраской.

## Технологии
- HTML5 / CSS3 / JavaScript  
- Three.js (3D рендеринг + OrbitControls)
- Space Grotesk (Google Fonts)
- Mobile-first адаптивный дизайн
